## Video Game Sales Analysis

### Business Problem

An online video game retailer aims to optimize its marketing strategy for the upcoming year by identifying the characteristics that drive commercial success. Using historical sales data, critic and user reviews, platform information, genres, and ESRB ratings, this project analyzes the factors associated with high-performing video games and provides data-driven recommendations for future marketing investments.

### Objective

The objective of this project is to explore historical video game sales and uncover patterns that influence a game's commercial performance. The analysis focuses on identifying profitable platforms, genres, and regional market preferences while evaluating the relationship between review scores and sales. Statistical hypothesis testing is also used to validate key assumptions about user ratings across platforms and genres.

### Dataset

The dataset contains historical video game sales up to 2016 and includes information such as:

* Game title
* Release year
* Platform
* Genre
* Publisher
* North America, Europe, Japan, and Other regional sales
* Global sales
* Critic scores
* User scores
* Number of critic and user reviews
* ESRB age rating

### Project Workflow

The project follows a complete data science workflow:

1. Data exploration and quality assessment.
2. Data cleaning and preprocessing.
3. Feature engineering by calculating global sales.
4. Exploratory Data Analysis (EDA) to identify trends across platforms, genres, and regions.
5. Regional market segmentation for North America, Europe, and Japan.
6. Correlation analysis between review scores and commercial performance.
7. Statistical hypothesis testing using independent two-sample t-tests.
8. Business insights and recommendations for future marketing campaigns.

### Key Business Questions

* Which platforms generate the highest revenue?
* Which genres consistently outperform others?
* How do regional preferences differ across major markets?
* Do critic and user reviews influence game sales?
* Which platforms show the strongest growth potential?
* Does the ESRB rating impact sales across different regions?
* Are user ratings significantly different between Xbox One and PC?
* Do Action and Sports games receive different average user ratings?

The final analysis provides actionable insights that can support marketing decisions by identifying high-potential platforms, genres, and target audiences while demonstrating a complete end-to-end exploratory data analysis workflow commonly used in real-world data science projects.
